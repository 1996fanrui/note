# 栈

## 介绍

- 关于“栈”，我有一个非常贴切的例子，就是一摞叠在一起的盘子。我们平时放盘子的时候，都是从下往上一个一个放；取的时候，我们也是从上往下一个一个地依次取，不能从中间任意抽出。**后进者先出，先进者后出**，这就是典型的“栈”结构。

    ![](./picture/栈原理.jpg)

- 栈是一种“操作受限”的线性表，**只允许在一端插入和删除数据**

- 当某个数据集合只涉及在一端插入和删除数据，并且满足后进先出、先进后出的特性，我们就应该首选“栈”这种数据结构。
  
- 栈的实现
    
    - 用数组实现的栈，我们叫作**顺序栈**
    
    - 用链表实现的栈，我们叫作**链式栈**

- 栈的操作

    - 入栈
    
    - 出栈

- 支持动态扩容的顺序栈
